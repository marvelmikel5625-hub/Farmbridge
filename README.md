# 🌾 FarmBridge

**Nigeria's Agricultural Marketplace**

Connect directly with verified farmers and buyers across Nigeria. Secure escrow payments via Paystack.

## 🚀 Features

- 🛒 **Marketplace** — Browse tractors, seeds, fertilizer, crops, livestock, and more
- 🔐 **KYC Verification** — Verified buyers and sellers you can trust
- 💳 **Secure Escrow** — Paystack-powered payments held safely until delivery
- 📊 **Seller Dashboard** — Manage listings, track orders, view earnings
- 📝 **Blog & Courses** — Farming guides, market insights, and agricultural training
- 📱 **WhatsApp Integration** — Connect directly with sellers
- 💰 **Subscription Plans** — Basic (Free), Pro, Premium, Business

## 🛠️ Tech Stack

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Charts:** Recharts
- **Forms:** React Hook Form + Zod
- **Backend:** Supabase (Auth, Database, Storage, Realtime)
- **Payments:** Paystack (Checkout, Subscriptions, Escrow)
- **Deployment:** Vercel

## 📦 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation
```bash
git clone https://github.com/marvelmikel5625-hub/Farmbridge.git
cd Farmbridge
npm install
```
